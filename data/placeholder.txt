# Data Directory Placeholder

This directory is used to store runtime data files generated by the Password Manager application, including:

1. `vault.db`: The SQLite database containing your encrypted credentials (saved in binary BLOB format).
2. `salt.bin`: A 16-byte random salt generated on first run, used for PBKDF2-HMAC-SHA256 key derivation.
3. `master.hash`: A SHA-256 hash of the master password, used solely to verify login attempts.

## Security & Git Policies

For security and privacy reasons, all runtime-generated security assets (`vault.db`, `salt.bin`, `master.hash`) are excluded from git tracking via the `.gitignore` file. This prevents sensitive credentials and local cryptographic keys from being committed to GitHub.

This placeholder file is kept in version control so that the `data/` directory is automatically tracked and created when the repository is cloned.
